# Olive フレキシブルペイ 損益分岐シミュレーター

本リポジトリは、三井住友カードの「プラチナプリファード」および「Olive フレキシブルペイ Visa Infinite」の複雑な還元構造について、生成AIを用いて詳細な比較検討を行い、その結果に基づき各種条件を比較しやすい資料（HTMLツール）を作成・公開するものです。

## プロジェクトの概要

クレジットカードの年会費免除条件やポイント還元率は非常に複雑ですが、個人の「年間決済額」「資産残高」「毎月のクレカ積立額」をプルダウンで選択するだけで、どのカードが一番お得になるかを即座に計算できる「基本シミュレーター（`olive.html`）」と、条件を変えた2つのパターンを横並びで比較できる「比較シミュレーター（`olive_comp.html`）」の2種類のツールを開発しました。

また、本ツールの基礎理論となる各カードの詳細な仕様比較、および最適化戦略に関する総合的な分析結果については、生成AIを活用して作成した以下のレポートにまとめています。
